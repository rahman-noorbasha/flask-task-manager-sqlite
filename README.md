# Flask Task Manager with SQLite

A task management web application built using Flask and SQLite.

## Features

- Add tasks
- Delete tasks
- Persistent database storage
- Browser-based interface

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS

## Run Locally

```bash
pip install flask
python app.py
```

Open:

http://127.0.0.1:5000

## Learning Outcomes

- Flask routing
- Form handling
- SQLite database operations
- SQL queries
- Dynamic rendering
